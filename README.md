# Django Blog Project

A simple and functional blog application built with **Django 6**. This project focuses on backend fundamentals, including CRUD operations, database management, and URL routing.

## 🚀 Features
- **Admin Panel:** Manage blog posts through Django's built-in admin interface.
- **Dynamic Routing:** Post detail pages using unique IDs.
- **Backend Logic:** Built using the Model-View-Template (MVT) architecture.
- **Clean URLs:** Optimized paths for better navigation.

## 🛠 Tech Stack
- **Python 3.14** (Development Version)
- **Django 6.0.3**
- **SQLite** (Default Database)
